Explain What A Class Is and It's Relation to Objects

Class:

  • A blueprint from which objects are created.
  • Combines the fields and methods into a single unit, but we will get to that later in the tutorial.
  • Supports polymorphism, inheritance, and also provide the concept of derived classes and base classes.

Declaration of class:

  • contains only a keyword class, followed by identifier(name) of the class.

Objects:

  • A basic unit of object-oriented programming and respresents real-life entities.
  • State: It is represented by attributes of an object. It also reflects the properties of an object.
  • Behavior: It is represented by the methods of an object. It also reflects the response of an object with other objects.
  • Identity: Gives a unique name to an object and enbales one object to interact with other objects.
