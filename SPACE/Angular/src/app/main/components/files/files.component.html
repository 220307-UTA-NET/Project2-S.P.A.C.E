File.WriteAllText() Method

The File class provides functionality for file handling in C#. The File.WriteAllText(path) method can be used to write some string to a file in the path path.

File.ReadAllText() Method

The File.ReadAllText() method can be used to read data from a file in the form of a string variable. The File.ReadAllText() method takes the path of the file as a parameter and returns the contents of the file in string data type.

StreamWriter Class

The StreamWriter class is used to write data to a stream in a particular encoding in C#. The StreamWrite.WriteLine() method can be used to write a string variable to a file.

StreamReader Class

The StreamReader class is used to read data from a stream in a particular encoding in C#. The StreamReader.ReadLine() method can be used to read string data from a file.
