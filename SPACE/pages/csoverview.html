Overview of C#

What is C#?

C# is an open source cross-platform object oriented programming language developed by Microsoft to create and use software components utilizing the .NET Framework to develop web applications, mobile applications, games, and others.

It shares common traits with other object oriented languages such as Java, C++, etc.

Why use C#?

It's among the most popular languages that have extensive documentation and development.

It has a concise structure and allows code to be packaged and reused.

It makes it easier to swtich between other OOP languages, such as Java.
