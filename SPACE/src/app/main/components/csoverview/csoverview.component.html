What is C#?

C# is an open-source cross-platform object oriented programming language developed by Microsoft to create and use software components utilizing the .NET Framework to develop web applications, mobile applications, games, and other resources.

It shares common traits with other object oriented languages such as Java

Why use C#?

  • It's among the most popular languages that have extensive documentation and development
  • It has a concise structure and allows code to be packaged and reused
  • Makes it easier to switch between other OOP languages such as Java
