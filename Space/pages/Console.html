Introducing Console.WriteLine, Console.Write,Console.Readline, and Console.Read

Console.WriteLine Method

Writes the specified data, followed by the current line terminator, to the standard output stream.

Console.WriteLine is used to print data along with printing the new line.

        
            public status Program{
                static void Main(string[] args)
                {
                    /* output:
                    Hello
                    World
                    */
                    Console.WriteLine("Hello!");
                    Console.WriteLine("World!");
                }
            }

Console.Write Method

Console.Write is used to print data without printing the new line.

Writes the text representation of the specified value or values to the standard output stream.

        
            public class Program{
                static void Main(string[] args)
                {
                    /* output:
                    Hello World!
                    */
                    Console.Write("Hello");
                    Console.Write("World!");
                }
            }

Console.ReadLine Method

This method is used to read the next line of characters from the standard input stream.

If the standard input device is the keyboard, the ReadLine method blocks until the users presses the Enter key.

If the standard input is redirected to a file, then this method reads a line of text from a file.

        
            public static void Main()
            {
                int age;
                string name;

                Console.WriteLine("Enter your name: ");

                // using the method.
                // typecasting not needed
                // as ReadLine returns string
                name = Console.ReadLine();
            }

Console.Read Method

Console.Read is used to read the next character from the standard input stream.

This method basically blocks its return when the user types some input characters. As soon as the user press ENTER key it terminates.

        
            static void Main(string[] args)
            {
                int x;
                Console.WriteLine("Enter your Character to get Decimal number");
                
                // using the method
                x = Console.Read();
                Console.WriteLine(x);
            }
